# SLIDE-PATTERN-team-members-grid

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** team-members-grid
**概要：** メンバーを4列×2行のカードで並べるチーム紹介レイアウト。各カードは円形写真プレースホルダ＋氏名＋役職＋一言。
**適したシーン：** 会社紹介・採用資料のチーム紹介、講座の講師陣紹介。4〜8名の顔ぶれ提示。

## Structure（構造）
- 4列グリッド × 最大2行（4〜8名）
  - カード: 円形写真 → 氏名 → 役職（アクセント色） → 一言

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 円形写真 | カード上部 | 実装時は img を円形クリップで差し込み（プレースホルダは人物アイコン） |
| 氏名 | 写真下 | 太字 |
| 役職 | 氏名下 | アクセント色の小さな太字 |
| 一言 | 最下 | 経歴・担当の1行 |

## Usage Guide（AIへの使い方）
- 4名以下なら1行にして写真を80pxに拡大してよい
- 写真が無い場合はアイコンのままでよい（無理に画像を作らない）
- 1名を深掘りする場合は profile-bio を使う
