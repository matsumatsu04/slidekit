## Overview

**パターン名：** positioning-map-scatter
**概要：** 縦横2軸の十字座標上に競合・商品・プランなどをドットとラベルで配置し、相対的な位置関係を見せる散布図（ポジショニングマップ）
**適したシーン：** 競合比較、商品ポートフォリオ、市場マッピングなど2軸で複数の対象の立ち位置を一望させたい場面

## Structure（構造）

コンテンツエリア中央に縦横の十字軸線を配置し、上下左右の四端に軸ラベルを置く。平面上には複数のドット（円）とラベルのペアを%座標で配置し、そのうち1点だけを強調表示することで注目させたい対象を際立たせる。

    structure:
      layout: single-plane
      axis:
        elements:
          - axis-line-vertical (中央)
          - axis-line-horizontal (中央)
          - axis-label-top
          - axis-label-bottom
          - axis-label-left
          - axis-label-right
      plot:
        type: scatter-dots
        max-items: 8
        elements:
          - dot-plain (複数、アウトライン円＋通常ラベル)
          - dot-accent (1点のみ、塗りつぶし円＋太字ラベル)

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 縦軸線 | コンテンツエリア中央（縦） | 横方向の評価軸の基準線 |
| 横軸線 | コンテンツエリア中央（横） | 縦方向の評価軸の基準線 |
| 軸ラベル（上/下/左/右） | コンテンツエリアの四端 | 各軸が何を評価しているかを示す |
| 通常ドット＋ラベル | 平面上（%座標指定） | 比較対象の相対位置を示す |
| 強調ドット＋ラベル | 平面上（%座標指定・1点のみ） | 注目させたい対象を際立たせる |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-positioning-map-scatterのレイアウトで、縦軸を[評価軸名]、横軸を[評価軸名]とし、[対象名]を複数配置、その中で[強調したい対象名]だけを強調表示してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- ドットは最大8個・座標は%指定とする
- 強調するドットは1点のみに絞る（複数点を強調すると訴求点がぼやける）
- ドット同士・ドットと軸ラベルが重ならないよう、座標はおおむね全体の15%〜85%の範囲に収める
- 軸ラベルは短い語句にとどめ、詳細な説明文は含めない
