# SLIDE-PATTERN-price-total-highlight

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾は
デザインシステム（SLIDEKIT-DESIGN.md）側の Frame で定義します。

## Overview
**パターン名：** price-total-highlight
**概要：** 上部に見積の内訳リスト（項目名・説明・金額の3ゾーン行、縦罫のみで区切る）、下部に「合計」を特大数字＋小さめの単位・税注記で大きく提示する金額提示レイアウト。
**適したシーン：** Web制作・受託案件の見積提示、料金プランの最終確認、提案書の金額サマリーページ、請求前の合意形成スライド

## Structure（構造）

```yaml
content-area:
  layout: vertical-stack
  sections:
    - breakdown-list:            # 上部 約55%
        rows: 3〜4               # 内訳行
        row:
          - item-name:           # 左: 項目名（太字）
          - vertical-divider:    # 細い縦罫1本（横罫・外枠は使わない）
          - item-desc:           # 中: 内容の短い説明
          - item-price:          # 右: 金額（右揃え・等幅感）
    - total-band:                # 下部 約45%
        - total-label:           # 「合計」ラベル
        - total-amount:          # 特大数字（アクセント色）
        - total-unit:            # 単位（円/税込等）は数字の40%前後のサイズ
        - tax-note:              # 税・支払条件の小さな注記
```

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| breakdown-row | 上部に縦積み（3〜4行） | 見積の内訳。項目名｜説明｜金額を縦罫1本で区切る（表の檻を作らない） |
| item-price | 各行の右端・右揃え | 個別金額。桁を揃えて比較しやすく |
| total-band | 下部中央 | 合計金額の提示ゾーン。上部と細い罫線で区切る |
| total-amount | total-band中央・特大 | 合計数字。数字:単位:注記 ≒ 10:3:1 のジャンプ率で一目で伝える |
| tax-note | 合計の下・小さく | 税区分・有効期限・支払条件などの補足 |

## Usage Guide（AIへの使い方）

プロンプト例：
「price-total-highlight パターンで見積スライドを作成。内訳は『デザイン｜TOP+下層4P｜40万円』『実装｜レスポンシブ対応｜30万円』『ディレクション｜進行・品質管理｜10万円』、合計80万円（税別）」

注意点：
- 内訳は最大4行まで。それを超える明細は line-item-table を使う（このパターンは"サマリー"用）
- 合計数字のみアクセント色。内訳の金額は黒のまま（強調は1箇所）
- 端数まで見せたい場合も、合計はカンマ区切りで1つの数字として大きく見せる
