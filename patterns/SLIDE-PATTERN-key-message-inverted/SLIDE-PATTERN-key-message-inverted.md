# SLIDE-PATTERN-key-message-inverted

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** key-message-inverted
**概要：** スライド全面をアクセント色で塗り、白抜きのキーメッセージを中央配置する反転配色レイアウト。背景トーンの切替でデッキにリズムを作る。
**適したシーン：** 章の結論・最重要メッセージ。白地スライドが続いた後の「ここが山場」の1枚。

## Structure（構造）
- 全面: アクセント色ベタ
- 中央縦積み: 英字ラベル → 短い白線 → キーメッセージ（最大2行）→ 補足（省略可）

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 英字ラベル | 中央上 | KEY MESSAGE等。字間広め |
| キーメッセージ | 中央 | 白太字・最大2行 |
| 補足 | 中央下 | 出典・根拠を小さく |

## Usage Guide（AIへの使い方）
- 1デッキ1〜2枚まで（乱用すると効果が消える）
- 白地版の key-message-single と使い分ける（通常は白地版、山場だけ反転版）
- 共通見出し（.sk-h）は付けない
