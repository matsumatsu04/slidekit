このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** phase-staircase-lshape
**概要：** フェーズごとに「水平太バー＋左端の垂直バー」のL字ライン＋見出し＋箇条書きを、左下→右上へ段違い配置する階段型ロードマップ。矢印・グラデ不要。
**適したシーン：** 中期計画・成長ロードマップ・導入ステップ。右肩上がりの世界観を線だけで表現したい時。

## Structure（構造）
- フェーズブロック × 3（左下 → 中央 → 右上に段違い配置）
  - L字ライン（水平5px＋垂直5px）
  - PHASEラベル → フェーズ名 → 箇条書き2行

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| L字ライン | 各ブロック上・左 | 階段の「段」を示す。アクセント色 |
| PHASEラベル | ライン下 | 英字＋番号 |
| フェーズ名・内容 | ラベル下 | 太字見出し＋箇条書き2行まで |

## Usage Guide（AIへの使い方）
- フェーズは3つが基本（4つの場合は幅を狭めて等間隔に再配置）
- 箇条書きは各2行まで（あふれる場合は vertical-step-flow へ）
