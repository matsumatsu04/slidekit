# SLIDE-PATTERN-cover-big-typography

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** cover-big-typography
**概要：** 装飾を排し、58pxの特大極太タイトル（キーワード1箇所だけアクセント色）で語るタイポグラフィ表紙。画像不要。
**適したシーン：** メッセージ性の強い資料・ビジョン系資料・ミニマル系デッキの表紙。言葉が主役の場面。

## Structure（構造）
- 左上: 英字ラベル\n- 中央左: 特大タイトル（900ウェイト・最大3行。1フレーズだけ .em でアクセント色）\n- 左下: 短い太線 → 会社名・日付

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 特大タイトル | 中央左 | 58px/900。主役 |
| アクセント語 | タイトル内 | 1フレーズだけ色を変える |
| メタ情報 | 左下 | 会社名・日付 |

## Usage Guide（AIへの使い方）
- アクセント色にするのは1フレーズだけ（2箇所以上に増やさない）
- タイトルが3行を超えるなら言葉を削る（サイズは下げない）
