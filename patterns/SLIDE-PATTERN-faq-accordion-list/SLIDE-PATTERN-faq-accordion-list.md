# SLIDE-PATTERN-faq-accordion-list

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** faq-accordion-list
**概要：** Q行（淡色帯＋Q丸バッジ）とA行を1セットの枠にまとめ、4問を縦に積むコンパクトFAQ。
**適したシーン：** 質問数が4〜5問で、1画面に一覧で収めたい場面。説明会・営業資料の巻末FAQ。

## Structure（構造）
- Q&Aセット×4（枠囲み）\n- Q行: Qバッジ（アクセント丸・白文字）＋質問（淡色帯）\n- A行: Aマーク＋回答1〜2行

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| Qバッジ | 各セット左上 | アクセント丸に白のQ |
| 質問行 | 淡色帯 | 太字の質問 |
| 回答行 | 白地 | 1〜2行の回答 |

## Usage Guide（AIへの使い方）
- 回答は2行以内に要約する（長い回答は faq-single-column へ）
- 質問は4問が基準（5問まで可）
