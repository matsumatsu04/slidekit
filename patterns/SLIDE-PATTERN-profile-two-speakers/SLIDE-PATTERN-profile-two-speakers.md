# SLIDE-PATTERN-profile-two-speakers

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** profile-two-speakers
**概要：** 登壇者・共同創業者など2名を、大きめの円形写真＋氏名＋肩書＋経歴の同一カード2枚で並べる2名紹介レイアウト。
**適したシーン：** セミナーの登壇者紹介、対談相手の紹介、共同創業者・共同代表の紹介。

## Structure（構造）
- 左右2カード（同一構造）
- カード: 円形写真96px → 氏名 → 肩書（アクセント） → 区切り線 → 経歴3〜4行

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 円形写真 | カード上 | 実装時はimgを円形クリップ |
| 氏名・肩書 | 写真下 | 肩書はアクセント色 |
| 経歴 | 下部 | 3〜4行 |

## Usage Guide（AIへの使い方）
- 1名だけなら profile-bio、4名以上なら team-members-grid を使う
- 写真が無い場合はアイコンのままでよい
