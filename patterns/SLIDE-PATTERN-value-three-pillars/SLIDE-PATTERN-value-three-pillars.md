# SLIDE-PATTERN-value-three-pillars

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** value-three-pillars
**概要：** MISSION / VISION / VALUE のチップ（アクセント枠）＋大きな文言を3段で縦に積む理念提示レイアウト。
**適したシーン：** 会社紹介・採用資料のミッション/ビジョン/バリュー。3つの理念・約束・方針を格調高く見せる場面。

## Structure（構造）
- 3行（区切りは細罫線）
- 各行: 枠付きチップ（英字ラベル） → 文言（太字21px） → 補足（省略可）

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| チップ | 各行左 | アクセント枠の英字ラベル |
| 文言 | チップ右 | この行の主役（太字） |
| 補足 | 文言下 | 1行の説明（省略可） |

## Usage Guide（AIへの使い方）
- 文言は各1行に収める（2行になるなら削る）
- ラベルはMISSION/VISION/VALUE以外（約束・行動指針等）でもよい。3段が基準（2〜4段まで可変）
