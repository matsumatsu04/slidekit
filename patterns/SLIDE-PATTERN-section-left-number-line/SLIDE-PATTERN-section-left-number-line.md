# SLIDE-PATTERN-section-left-number-line

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。中扉（セクション区切り）パターンのため共通見出し（sk-head）は対象外です。

## Overview

**パターン名：** section-left-number-line
**概要：** 白地の背景に、左側に特大の章番号、右側に章タイトルとアクセント色の短い線を横並びに配置する、余白を活かしたミニマルな中扉レイアウト。
**適したシーン：** プレゼンの章区切り、セクション導入。落ち着いたトーンで章の切り替えを示したい場面。

## Structure（構造）

スライド全体（タイトルエリアなし）を横方向中央揃えで使い、左に特大の章番号、右に章タイトルと短いアクセント線を縦に積んだブロックを並べる。

    structure:
      layout: full-slide-horizontal-pair (no title area)
      left:
        element: big-number (120px級, var(--sk-soft)色)
      right:
        alignment: vertical-stack
        elements:
          - section-title (32px, bold, #333)
          - accent-line (56px×3px, 角丸, var(--sk-accent))

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 特大の章番号 | スライド左側 | 章の位置づけを大きな数字で象徴的に示す（淡いトーンで主張しすぎない） |
| 章タイトル | 右側上段 | この章の主題を大きく太い文字で表示 |
| アクセント線 | 章タイトルの下 | 短い線でタイトルを引き締め、アクセント色を一箇所だけ効かせる |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-section-left-number-lineのレイアウトで、章番号「03」、章タイトル「[タイトル]」の中扉スライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 章番号は淡いトーン（`--sk-soft`）で大きく表示し、主役の章タイトルを邪魔しない
- 章番号は「01」「02」のようなゼロパディング2桁が定番
- 章タイトルは2行以内に収まる長さが望ましい
- アクセント線はこのスライドで唯一の強調要素として使う（他に色を足さない）
