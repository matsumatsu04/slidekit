# SLIDE-PATTERN-line-item-table

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** line-item-table
**概要：** 「内訳」見出しの下に、項目・単価・数量・単位・金額の明細テーブルのみを配置するシンプルな一覧レイアウト。合計行は持たず、各行に余白を取って読みやすくする。背景装飾は用いない。
**適したシーン：** 追加オプション・料金メニュー・参考価格などの一覧提示、合計を出さない明細表。

## Structure（構造）

コンテンツエリア上部に小見出し「内訳」（短い下線アクセント付き）を置き、その下に明細テーブルを全幅で配置する。ヘッダー行を反転で強調し、明細行は行間を広めにとってゆったり見せる。合計行は設けない。背景はプレーン（ドット等の装飾なし）。

    structure:
      layout: breakdown-table-only
      background: plain
      top:
        - subheading (with short underline)   # 例:「内訳」
      body:
        table:
          columns: [項目, 単価, 数量, 単位, 金額]
          header-row: emphasized (filled)
          item-rows: 行間広め / 合計行なし

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 内訳見出し | 上部左 | 一覧セクションの小見出し（短い下線アクセント） |
| 明細テーブル | 中央〜下部 | 項目・単価・数量・単位・金額の5列。ヘッダー行を反転で強調 |
| 明細行 | テーブル本体 | 各項目を1行ずつ。行間を広めにとって読みやすくする（合計行は持たない） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-line-item-table のレイアウトで、内訳テーブル（項目・単価・数量・単位・金額）を作成してください。合計行は不要です。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 列は「項目・単価・数量・単位・金額」の5列が基本。数値は右寄せ（または中央揃え）で桁を揃える。
- ヘッダー行は反転（塗り）で強調する。塗り色はSLIDE.mdのブランドカラー（濃色）を使う。
- 合計を示したい場合は estimate-amount-table を使う（このパターンは合計行を持たない）。
- 明細行は4〜6行程度。行間を広めにとって余白で読みやすくする。
- 背景に装飾（ドット等）は入れない。
