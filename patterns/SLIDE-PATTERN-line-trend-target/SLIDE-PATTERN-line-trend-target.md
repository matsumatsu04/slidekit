# SLIDE-PATTERN-line-trend-target

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** line-trend-target
**概要：** 右肩上がりの折れ線（アクセント色）と破線の目標ライン、終点だけ差し色の大きめドット＋数値で「目標にどこまで迫ったか」を示すトレンドグラフ。
**適したシーン：** 達成率・継続率・進捗などの推移報告。目標との距離感を1枚で伝えたい場面。

## Structure（構造）
- 上部: 破線の目標ライン＋ラベル
- 中央: 折れ線（5点）。終点のみ差し色ドット＋数値
- 下部: ベースライン＋期ラベル

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 目標ライン | 上部・破線 | 基準線（N2④） |
| 折れ線 | 中央 | アクセント色4px |
| 終点ドット＋数値 | 右上 | 差し色の1点強調 |
| 期ラベル | 下部 | Q1〜Q5等 |

## Usage Guide（AIへの使い方）
- ポイント座標（polyline points）は実データ比で調整
- 点は4〜6個まで。多い場合は間引く
- 下降トレンドで使う場合は目標ラインを下に置き直す
