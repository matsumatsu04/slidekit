# SLIDE-PATTERN-section-tag-progress

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** section-tag-progress
**概要：** 上部に全章のタグ型ナビ（現在章だけアクセント塗り）を並べ、中央左に特大の章番号＋章タイトルを置く現在地つき中扉。
**適したシーン：** 長尺デッキの章区切り。読み手に全体の中の現在地を示しながら章を切り替えたい場面。

## Structure（構造）
- 上部: 章タグ×5（現在章のみアクセント塗り＋白文字）
- 中央左: 淡色の特大章番号 → 章タイトル → サブタイトル

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 章タグ列 | 上部 | 全章を等幅で並べ現在章のみ塗る |
| 章番号 | 中央左 | 淡色の特大数字 |
| 章タイトル | 番号下 | 太字・最大2行 |

## Usage Guide（AIへの使い方）
- 章数は4〜6個まで（7個以上はタグの文字が潰れるため章名を短縮する）
- current タグは章の進行に合わせて移動する
