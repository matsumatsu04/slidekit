# SLIDE-PATTERN-estimate-amount-table

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** estimate-amount-table
**概要：** 上部に「金額」ラベルの反転バッジと合計金額を大きく示し、その下に「内訳」見出しと明細テーブル（項目・単価・数量・単位・金額）を配置する見積レイアウト。テーブルのヘッダー行と合計行を反転（塗り）で強調する。背景装飾は用いない。
**適したシーン：** 概算見積・料金提示・費用内訳の説明スライド、提案書の見積ページ。

## Structure（構造）

コンテンツエリアの上段に「ラベルバッジ＋大型金額」を横並びで置き、中段に小見出し「内訳」（短い下線アクセント付き）、下段に明細テーブルを配置する。テーブルはヘッダー行を反転、最下部に合計行を反転で強調し、合計金額に視線を誘導する。背景はプレーン（ドット等の装飾なし）。

    structure:
      layout: amount-headline + breakdown-table
      background: plain
      top:
        - amount-badge (filled label box)     # 例:「金額」
        - amount-figure (extra-large)         # 例:「50,000円」
      middle:
        - subheading (with short underline)   # 例:「内訳」
      body:
        table:
          columns: [項目, 単価, 数量, 単位, 金額]
          header-row: emphasized (filled)
          item-rows: left-aligned label + right/center-aligned numbers
          total-row: emphasized (filled)      # 例:「合計」

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 金額バッジ | 上段左 | 「金額」等のラベルを反転ボックスで表示 |
| 金額（大） | バッジの右 | 合計金額を特大で表示し主役にする |
| 内訳見出し | 中段左 | 明細セクションの小見出し。テキストの「下」に短い太線（下線バー）を敷く |
| 明細テーブル | 下段 | 項目・単価・数量・単位・金額の5列。ヘッダー行を反転で強調 |
| 合計行 | テーブル最下行 | 合計を反転（塗り）で強調し、上段の金額と一致させる |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-estimate-amount-table のレイアウトで、金額「[合計金額]」、内訳テーブル（項目・単価・数量・単位・金額）と合計行を作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 列は「項目・単価・数量・単位・金額」の5列が基本。数値は右寄せ（または中央揃え）で桁を揃える。
- ヘッダー行と合計行は反転（塗り）で強調する。塗り色はSLIDE.mdのブランドカラー（濃色）を使う。
- 上段の大型金額と合計行の金額は必ず一致させる。
- 明細行は5〜6行程度まで。多い場合は行を間引くか別スライドに分ける。
- 背景に装飾（ドット等）は入れない。
