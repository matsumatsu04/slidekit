# SLIDE-PATTERN-area-growth-curve

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** area-growth-curve
**概要：** 右肩上がりの成長カーブを「線＋薄い面（塗り）」で描き、右上に最終値を大きく添えるエリアグラフ（SVG）。
**適したシーン：** 売上・ユーザー数の成長ストーリー。棒より滑らかに「勢い」を演出したい表紙級の1枚。

## Structure（構造）
- 面: アクセント色18%の薄塗りポリゴン\n- 線: アクセント色4pxの折れ線（5点）\n- 右上: 最終値（大きな数字）＋補足\n- 下部: ベースライン＋期ラベル

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 面（塗り） | 線の下 | color-mixの薄色でパレット連動 |
| 成長線 | 中央 | アクセント色 |
| 最終値 | 右上 | 結論の数字 |

## Usage Guide（AIへの使い方）
- ポイント座標は実データ比で調整
- グリッド線・目盛りはあえて置かない（勢い重視の演出用。正確な比較は棒グラフで）
