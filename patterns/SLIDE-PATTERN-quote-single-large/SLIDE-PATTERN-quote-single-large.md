# SLIDE-PATTERN-quote-single-large

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** quote-single-large
**概要：** 特大の引用符＋2〜3行の引用文を主役にし、下部に発言者（円形写真＋氏名・肩書）を添える1名大型引用レイアウト。
**適したシーン：** 顧客の声・推薦コメント・受講生の感想を「1つだけ深く」見せる場面。導入事例の冒頭。

## Structure（構造）
- 左上: 特大引用符（淡色）
- 中央: 引用文（太字24px・最大3行）
- 下部: 円形写真＋氏名・肩書

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 引用符 | 左上 | 淡色の特大装飾 |
| 引用文 | 中央 | このスライドの主役 |
| 発言者 | 左下 | 写真＋氏名＋肩書 |

## Usage Guide（AIへの使い方）
- 引用文は3行以内に要約する（長い場合は最も強い一文だけ残す）
- 複数の声を並べたい場合は testimonial-quote-grid を使う
