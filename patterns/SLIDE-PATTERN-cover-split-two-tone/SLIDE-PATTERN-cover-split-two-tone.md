# SLIDE-PATTERN-cover-split-two-tone

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** cover-split-two-tone
**概要：** 左42%をアクセント色ベタ（白抜きタイトル）、右58%を白（余白＋右下メタ情報）にする2トーン分割の表紙。画像素材が不要な表紙の定番。
**適したシーン：** 提案書・報告書・勉強会資料の表紙。画像を使わずに締まった表紙を作りたい場面。

## Structure（構造）
- 左42%: アクセント色ベタ: 種別ラベル → タイトル（白・最大2行） → サブ
- 右58%: 白地: 右上に円の線装飾1つ・右下に会社名・日付

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 左パネル | 左42% | アクセント色ベタ＋白抜きタイトル |
| 種別ラベル | タイトル上 | PROPOSAL等の英字 |
| メタ情報 | 右下 | 会社名・日付 |
| 円装飾 | 右上 | 細線の円1つだけ（増やさない） |

## Usage Guide（AIへの使い方）
- 画像背景の表紙（cover-photo系）と使い分け。素材なしで即使えるのがこのパターンの価値
- 右の余白は埋めない（余白が上質さを作る）
