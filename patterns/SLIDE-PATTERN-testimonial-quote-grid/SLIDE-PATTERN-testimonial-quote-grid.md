# SLIDE-PATTERN-testimonial-quote-grid

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** testimonial-quote-grid
**概要：** 3列×2行＝6枚の引用カードを並べ、各カードに引用符グリフ・お客様の声本文・属性（会社名／役職）を配置するお客様の声グリッドレイアウト
**適したシーン：** 導入事例・お客様の声の一覧提示、複数の推薦コメントをまとめて見せたい場面、信頼性訴求のセクション

## Structure（構造）

コンテンツエリア全体を3列×2行＝6枚のカードグリッドに分割する。各カードは上部に大きめの引用符グリフ、中央に「声」の本文（2〜3行）、下部に区切り線を挟んで会社名・役職の属性テキストを縦に積んで配置する。カードは薄枠のみで面の塗りや影は持たない。

    structure:
      layout: grid-of-cards
      columns: 3
      rows: 2
      card-count: 6
      card:
        top:
          elements: [quote-glyph]
        middle:
          elements: [testimonial-body]
        divider: horizontal-line
        bottom:
          elements: [attribution-text]
        style: thin-border-only (no shadow, no fill)

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 引用符グリフ | 各カード上部 | 「これは引用・声である」ことを視覚的に示すアクセント記号 |
| お客様の声本文 | カード中央 | 導入後の変化・感想を語る2〜3行のコメント |
| 区切り線 | 本文と属性の間 | コメント本文と発言者情報を視覚的に区切る |
| 属性テキスト（会社名／役職） | カード下部 | 発言者の所属・役職を示し、声の信頼性を補強する |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-testimonial-quote-gridのレイアウトで、お客様の声6件（各社名・役職・コメント）を3列×2行のカードに並べてください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 本文は2〜3行を目安にする。長くなりすぎるとカード内で属性テキストとの間隔が詰まるため、要約して収める
- カードに影・塗りつぶし背景は付けない（薄枠のみで統一する）
- 6件に満たない場合でも空カードは作らず、件数に応じて列数の少ない別パターン（3枚構成のカードグリッド等）を検討する
- 引用符グリフは装飾要素であり、本文の一部として読ませない（サイズを大きめ・色を薄めにして視覚的アクセントに留める）
