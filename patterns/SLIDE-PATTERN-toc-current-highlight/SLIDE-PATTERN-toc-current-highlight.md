# SLIDE-PATTERN-toc-current-highlight

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** toc-current-highlight
**概要：** 目次リストの「現在の章」だけをアクセント色の帯＋白文字＋NOWバッジで塗り、他章はグレーで沈める現在地ナビ型の目次。
**適したシーン：** 章の切り替わりで目次を再掲し「いまここ」を示す場面。長尺デッキの迷子防止。

## Structure（構造）
- 英字ラベル（AGENDA）
- 章リスト5行（番号＋章名）。現在章のみアクセント帯＋白文字＋NOWバッジ

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 現在章の帯 | リスト中 | アクセント塗り＋白文字で現在地を示す |
| 他の章 | 上下 | グレー文字で沈める |
| NOWバッジ | 現在章右端 | 白枠の小バッジ |

## Usage Guide（AIへの使い方）
- 章の進行に合わせて current クラスを移動して使い回す
- 初出の目次には toc系の通常パターンを使い、再掲時にこのパターンへ切り替える
