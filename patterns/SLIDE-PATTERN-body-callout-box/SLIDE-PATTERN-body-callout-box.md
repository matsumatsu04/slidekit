# SLIDE-PATTERN-body-callout-box

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** body-callout-box
**概要：** 本文の説明3〜4行の下に、左アクセントバー＋淡色地の「POINTボックス」で要点を浮き上がらせる文章系レイアウト。
**適したシーン：** 規約・前提・考え方の説明など文章が主体のページで、結論だけは確実に読ませたい場面。

## Structure（構造）
- 上: 本文（行間2.0・3〜4行）\n- 中: POINTボックス（左4pxアクセントバー＋淡色地＋太字）\n- 下: 注記（省略可）

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 本文 | 上 | 文脈の説明 |
| POINTボックス | 中央 | このスライドの結論 |
| 注記 | 下 | 出典・例外事項 |

## Usage Guide（AIへの使い方）
- POINTボックスは1つだけ（2つ以上に増やすなら numbered-list-with-body へ）
- 本文は4行以内。長い場合は2枚に分ける
