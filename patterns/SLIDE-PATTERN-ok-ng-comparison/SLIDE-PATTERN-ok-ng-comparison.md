# SLIDE-PATTERN-ok-ng-comparison

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** ok-ng-comparison
**概要：** 左にNG（グレー系＋×）、右にOK（アクセント＋✓）を同じ構造のカードで並べる対比レイアウト。テイストを揃えて「対立だけ」を作る。
**適したシーン：** ビフォーアフター・推奨/非推奨・誤解と正解の対比。ノウハウ解説や提案の説得パート。

## Structure（構造）
- 左右2カード（同一構造）
  - ヘッダー: 丸マーク（×/✓）＋タイトル
  - 本文: マーク付きの行 × 3
- NG側=グレー系固定（意図的にパレット非連動）、OK側=アクセント色

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| NGカード | 左 | グレーの枠・文字・×で「避けるべき」を示す |
| OKカード | 右 | アクセントの枠・✓で「推奨」を示す |

## Usage Guide（AIへの使い方）
- 必ずNGを左・OKを右に置く（視線の流れ＝改善の流れ）
- 行数は左右で揃える（2〜4行）
- NG側にアクセント色を使わない（対比が濁る）
