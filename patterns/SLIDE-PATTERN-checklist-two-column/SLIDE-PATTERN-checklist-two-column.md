# SLIDE-PATTERN-checklist-two-column

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** checklist-two-column
**概要：** チェックボックス付きの項目（見出し＋補足）を左右2カラム×4行で並べるチェックリスト。
**適したシーン：** 準備物・対応範囲・提供内容・確認事項の一覧。6〜8項目を「もれなく」見せたい場面。

## Structure（構造）
- 2カラム × 4行（最大8項目）
- 各行: チェックボックス（アクセント枠＋✓） → 見出し → 補足（省略可）

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| チェックボックス | 各行左 | アクセント色の枠と✓ |
| 見出し | ボックス右 | 項目名（太字） |
| 補足 | 見出し下 | 1行の説明（省略可） |

## Usage Guide（AIへの使い方）
- 項目は6〜8個。5個以下なら summary-three-points や icon-left-text-list を検討
- 「含まれないもの」を並べる場合は ok-ng-comparison を使う
