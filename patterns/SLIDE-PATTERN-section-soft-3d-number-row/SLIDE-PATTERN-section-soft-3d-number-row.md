# SLIDE-PATTERN: section-soft-3d-number-row

このファイルはスライドのレイアウトパターン定義書です。背景に画像アセット（assets/covers/）を使用するセクション中扉パターンで、テキストはHTMLとして画像の上に載せます。

---

## 概要

半透明3Dオブジェクトの淡色背景の左中央に、特大の章番号と章タイトル（＋短いアクセント線）を横並びに配置するIT/SaaS系のセクション中扉レイアウト。

---

## Elements

| 要素 | 配置 | 役割 |
|---|---|---|
| 背景 | 全面（/assets/covers/cover-bg-soft-3d.jpg） | 半透明3Dオブジェクトの淡色素材。別アセットに差し替え可 |
| 章番号 | 左・行の先頭 | 特大・淡色で章の切り替えを示す |
| 章タイトル | 章番号の右 | 濃色太字・最大2行 |
| アクセント線 | タイトル直下 | 短い水平線で締める |

---

## Usage Guide

- テキストは左半分の余白に収める（右側の3Dオブジェクト群に被せない。max-width目安560px）
- 表紙 cover-soft-3d-left と同じ背景を使うと、表紙〜中扉でトーンが揃ったデッキになる
- 白地ミニマル版の section-left-number-line と同構図。画像背景で世界観を持たせたいときにこちらを使う
- 背景に画像を焼き込むため、ギャラリーの色ティントは適用されない
