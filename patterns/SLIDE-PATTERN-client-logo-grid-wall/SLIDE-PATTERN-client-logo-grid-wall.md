# SLIDE-PATTERN-client-logo-grid-wall

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** client-logo-grid-wall
**概要：** コンテンツエリア上部に短い見出し行を置き、その下に5列×4行＝20個の均等な矩形セルを並べて導入企業名を一覧表示するロゴウォールレイアウト
**適したシーン：** 導入実績・取引先一覧、パートナー企業紹介、対応業界一覧など「数の多さ・網羅感」を訴求したい場面

## Structure（構造）

コンテンツエリアを縦に2段へ分割する。上段は見出し行（見出しテキストと補足の件数表記を左右に配置）、下段はコンテンツエリアの残り全体を占める5列×4行のグリッドで、各セルは均等な矩形として中央にテキストを配置する。

    structure:
      layout: heading-row + grid
      heading:
        elements: [heading-text, count-supplement]
        align: space-between (baseline)
      grid:
        columns: 5
        rows: 4
        cell-count: 20
        cell:
          border: thin
          content-align: center (horizontal + vertical)
          elements: [company-name-placeholder]

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 見出しテキスト | 上段・左 | このグリッドが何の一覧かを示す短いラベル |
| 件数補足 | 上段・右 | 「全◯社」等の数の訴求を添える補足テキスト |
| グリッド（5列×4行） | コンテンツエリア下段全体 | 20件の企業名を均等なセルで一覧表示する |
| 各セル | グリッド内20マス | 薄枠の矩形1マス＝企業名1件。中央揃えでテキストを配置する |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-client-logo-grid-wallのレイアウトで、導入企業20社の名前を5列×4行のグリッドに並べてください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 20マスすべてを埋めることが前提。企業数が20に満たない場合は空セルを作らず、行数を減らすか別パターン（少数枚数向けのカードグリッド等）を検討する
- 実際のロゴ画像は使わず、企業名テキスト（またはロゴ差し替え前提のプレースホルダー）で統一する
- 各セルのテキストは1〜2行に収まる長さ（社名の長短が揃わない場合はフォントサイズを一段階縮小して調整）
- 件数補足（「全20社」等）は省略しても成立するが、入れる場合は見出しと同じ行に収める
