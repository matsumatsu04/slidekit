# SLIDE-PATTERN-waterfall-bridge-chart

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** waterfall-bridge-chart
**概要：** 開始値から複数の増減要因を経て終了値に至る過程を、浮遊する棒（ブリッジ）でつなぐウォーターフォールチャート
**適したシーン：** 売上・利益・コスト等の期首から期末までの内訳変動要因を分解して見せたい場面（増減要因分析、ブリッジ分析）

## Structure（構造）

コンテンツエリアを1つのチャートエリアとし、横方向に「開始値バー」「増減の浮動バー（複数）」「終了値バー」を等間隔のカラムとして並べる。開始値バーと終了値バーはチャートエリアの全高を基準に0から積み上げる固定バー、中間の増減バーは前バーの累積値の位置から浮いた状態で配置する。各バーの直上に数値ラベル、バー間には水平の接続線を渡し、下部に系列名（期首・要因名・期末）を並べる。

    structure:
      layout: single-column
      chart-area:
        type: waterfall-bars
        columns: 5 (可変、開始+増減3〜4本+終了)
        elements:
          - start-bar (固定・全高基準)
          - floating-bar (増加 / 減少、複数本)
          - end-bar (固定・全高基準・強調)
          - value-label (各バー直上)
          - connector-line (バー間の水平線、複数)
      labels-row:
        type: category-labels
        elements:
          - series-label (各カラム直下、例：期首/要因A/要因B/要因C/期末)

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 開始値バー | チャートエリア左端・0から積み上げ | 起点となる値を全高基準で表示 |
| 増加バー | 中間カラム・前バー上端から浮いた位置 | プラス要因の増分を薄い塗りで表示 |
| 減少バー | 中間カラム・前バー上端から浮いた位置 | マイナス要因の減分を枠線のみで表示（増加バーと区別） |
| 終了値バー | チャートエリア右端・0から積み上げ | 最終値を強調色で全高基準表示 |
| 数値ラベル | 各バーの直上 | そのバーが表す値・増減量を数値で明示 |
| 接続線 | バー間（前バー上端/下端〜次バー開始点） | 累積値のつながりを視覚的に示す |
| 系列名ラベル | チャートエリア下部・各カラム直下 | 開始点・各要因・終了点の名称 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-waterfall-bridge-chartのレイアウトで、期首[数値]から[要因名]の増減を経て期末[数値]に至る内訳を、開始値バー・浮動バー・終了値バーで表現してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 各バーの縦位置（累積値の高さ）は自動計算に頼らず、開始値・各増減量・終了値の整合性を事前に計算してから配置する
- 増加バーと減少バーは塗り/枠線で明確に区別する
- 中間バーは3〜4本程度に絞り、それ以上は視認性が落ちるため避ける
- 終了値バーのみ強調色を使い、他のバーと差をつける
