# SLIDE-PATTERN-toc-section-cards

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** toc-section-cards
**概要：** 章をカード（上辺アクセントライン＋大番号＋章名＋一言説明＋開始ページ）で4枚横並びにするカード型目次。
**適したシーン：** 章ごとの内容を一言添えて予告したい目次。ワークショップ・研修資料の冒頭。

## Structure（構造）
- 見出し行（CONTENTS＋タイトル）\n- 4カード: 上辺3pxライン → 大番号（淡色） → 章名 → 一言説明 → 開始ページ

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 大番号 | カード上 | 淡色30px |
| 章名 | 中央 | 太字 |
| 一言説明 | 章名下 | その章で得られること |
| 開始ページ | 下 | p.n〜 |

## Usage Guide（AIへの使い方）
- 章は4つまで（5章以上は toc-two-column へ）
- 一言説明は「何が得られるか」で書くと読み手が乗る
