# SLIDE-PATTERN-left-context-data

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** left-context-data
**概要：** 左25〜30%を淡色の「文脈固定」エリア（章番号・見出し・要点）にし、右にデータ・図表を置く左右分割レイアウト。
**適したシーン：** データや図表が主役のページ。連続するデータページで左レイアウトを固定し、読み手を迷わせたくない時。

## Structure（構造）
- 左28%（淡色面）: 章番号 → 見出し → 短いアクセント線 → 要点テキスト
- 右72%: メインデータ領域＋補足データ領域（縦2分割。補足は省略可）

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 章番号 | 左上 | 現在地の提示 |
| 見出し・要点 | 左面 | このページの結論を先に示す |
| データ領域 | 右 | グラフ・表・図解の実体（他パターンの部品を流用可） |

## Usage Guide（AIへの使い方）
- 連続ページで使う場合、左エリアの構成（番号・見出し・要点）は全ページで揃える
- 共通見出し（.sk-h）とは併用しない（左エリアが見出しを兼ねる）
