# SLIDE-PATTERN-bar-diff-bracket

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** bar-diff-bracket
**概要：** 2本の棒（前=グレー・後=アクセント）の頂点差を破線ガイド＋ブラケット＋差分バッジ（差し色）で強調する比較グラフ。
**適したシーン：** 導入前後・施策前後・自社と平均の「1点の差」を最も分かりやすく見せたい場面。

## Structure（構造）
- ベースライン上に棒2本（左=比較元グレー・右=比較先アクセント）
- 比較元の頂点から破線ガイドを右へ
- 右端にブラケット＋差分バッジ（--sk-accent2）

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 比較元バー | 左 | グレー固定（意図的にパレット非連動） |
| 比較先バー | 右 | アクセント色・数値も大きく |
| 破線ガイド | 2本の頂点間 | 差の起点を示す |
| 差分バッジ | 右端 | +83%等の結論（差し色） |

## Usage Guide（AIへの使い方）
- 棒の高さ（38%/70%）は実データ比で調整する
- 差分バッジの数値が結論。本文で同じ数字を繰り返さない
- 3本以上の比較は bar-actual-forecast を使う
