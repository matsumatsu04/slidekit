# SLIDE-PATTERN-gantt-schedule-bands

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** gantt-schedule-bands
**概要：** 制作工程のスケジュールを示すシンプルなガントチャート。上部に週／月の目盛り行、左に工程名の列（5行程度）、各行に期間を示す横帯（アクセント色の矩形）を配置する。矩形と罫線のみで構成し、複雑なガントツールの見た目を再現しない。
**適したシーン：** Web制作の提案・納品資料での制作スケジュール提示、プロジェクト工程表、納期の合意形成

## Structure（構造）

コンテンツエリアをグリッドで構成する。1列目は工程名ラベル（左寄せ）、2列目以降は期間の目盛り（週または月、4〜8列）。最上段は目盛りのヘッダー行（下罫線つき）。各工程行では、開始〜終了の目盛り列にまたがる横帯（塗り矩形）を1本置く。行間は薄い横罫線、目盛り列の境界は薄い縦罫線（または点線）で示す。

    structure:
      layout: grid (label-column + time-columns)
      header-row:
        cells: 4-8 (週/月の目盛りラベル, centered)
        divider: bottom rule
      rows: 4-6 (工程)
      row:
        left: process-name (bold, left-aligned)
        band:
          type: filled rectangle (1 per row)
          span: start-column to end-column
      gridlines:
        horizontal: thin rule between rows
        vertical: thin (or dashed) rule at column boundaries

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 目盛り行 | 最上段 | 週（1週目〜）または月（◯月〜）の時間軸ラベル |
| 工程名 | 各行左端・太字 | 要件定義／デザイン／実装／テスト／公開などの工程ラベル |
| 期間帯 | 各行の目盛り列上 | 工程の開始〜終了期間を示す塗り矩形（1行に1本） |
| 縦罫線 | 目盛り列の境界 | 期間の区切りを示す薄いガイド線 |
| 横罫線 | 行間 | 工程行の区切り |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-gantt-schedule-bands のレイアウトで、[要件定義／デザイン／実装／テスト／公開] の5工程を、[6週間] の目盛りでスケジュール表にしてください。各工程の期間は [◯週目〜◯週目] です。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 工程は4〜6行、目盛りは4〜8列まで。それ以上細かい場合は工程を統合する（詳細WBSはこのスライドの役割ではない）。
- 帯は1行に1本。分割帯・マイルストーン菱形・依存矢印は使わない（崩れる原因になる）。
- 帯の塗りはアクセント色1色で統一する。強調したい工程（例：公開）だけ濃度を変えるのは可（1箇所まで）。
- 帯の開始・終了は目盛り列の境界にスナップさせる（0.5週などの中途半端な位置にしない）。
- 目盛りラベルは「1週目/2週目…」「◯月」など全列で書式を揃える。
