# SLIDE-PATTERN-bar-actual-forecast

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** bar-actual-forecast
**概要：** 実績の棒グラフはグレー、最終期（予想・計画）だけアクセント色＋特大数値＋「予想」タグで強調する成長グラフ。右上に前年比バッジ（差し色）。
**適したシーン：** 売上・会員数などの成長トレンド提示。事業計画・決算説明・ピッチの「伸びている」1枚。

## Structure（構造）
- chart-area: 棒5本（左4本=実績グレー・右1本=予想アクセント）＋ベースライン
  - 実績列: 数値（小・グレー）＋バー
  - 予想列: 「予想」タグ＋特大数値＋アクセントバー
  - 右上: 前年比バッジ（差し色 --sk-accent2）
- axis-labels: 年度ラベル行

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 実績バー | 左4本 | グレー固定（意図的にパレット非連動） |
| 予想バー | 右端 | アクセント色。1本だけ塗って視線を集める |
| 特大数値 | 予想バー上 | ジャンプ率で結論を示す |
| 前年比バッジ | 右上 | 差し色の1点強調 |

## Usage Guide（AIへの使い方）
- 実績バーの高さは実データ比で調整（クラスb1〜b4のheight%）
- 「実績=グレー・計画=アクセント」の描き分けは崩さない（このパターンの本質）
- 本数は4〜6本まで。多い場合は間引く
