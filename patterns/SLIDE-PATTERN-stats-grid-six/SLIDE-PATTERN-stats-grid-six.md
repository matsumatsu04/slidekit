# SLIDE-PATTERN-stats-grid-six

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** stats-grid-six
**概要：** 「数字で見る◯◯」の定番。ラベル＋大きな数値＋補足のセルを3×2グリッドで並べ、細い罫線で区切る。
**適したシーン：** 会社紹介・採用資料の「数字で見る当社」、サービスの実績サマリー。

## Structure（構造）
- 3列×2行のグリッド（内側だけ細罫線）
- 各セル: ラベル → 数値（アクセント44px）＋単位 → 補足

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| ラベル | セル上 | 指標名 |
| 数値 | セル中央 | アクセント色の大きな数字 |
| 補足 | セル下 | 基準日・注記など |

## Usage Guide（AIへの使い方）
- 項目は6個ちょうどが最も美しい（4個なら2×2に組み替え）
- 数値の桁数が長い場合はフォントサイズを40px程度まで下げてよい
