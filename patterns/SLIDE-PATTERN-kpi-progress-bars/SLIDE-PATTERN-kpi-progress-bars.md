# SLIDE-PATTERN-kpi-progress-bars

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** kpi-progress-bars
**概要：** 指標名＋達成値（大きな数字）＋横プログレスバーのセットを3段積む進捗ダッシュボード。
**適したシーン：** KPIの達成状況報告・OKR進捗・キャンペーン途中経過。目標に対する現在地を直感的に見せる場面。

## Structure（構造）
- 3段（指標ごと）\n- 各段: 指標名｜達成値（アクセント大）＋目標 → プログレスバー（淡色トラック＋アクセント塗り） → 補足

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 達成値 | 右上 | アクセント色24px |
| バー | 中央 | widthが達成率そのもの |
| 補足 | バー下 | 前月比・注記 |

## Usage Guide（AIへの使い方）
- バーのwidth（%）と数値を必ず一致させる
- 指標は3つまで（4つ以上は goal-kgi-kpi-dashboard へ）
