# SLIDE-PATTERN-weekly-schedule-grid

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** weekly-schedule-grid
**概要：** カレンダー型の週間スケジュール表。左端に時刻の目盛り列、上部に月〜金の曜日ヘッダー帯を置き、各曜日列の中に高さの異なる予定ブロックを縦に配置する
**適したシーン：** 週間の稼働イメージ・標準スケジュールの提示、業務時間割の共有、コンサル/講座の週間サポート体制の説明

## Structure（構造）

コンテンツエリアをCSS Gridで構成する。1列目は時刻目盛り（固定幅）、2〜6列目は月〜金の曜日列（等幅）。最上段は曜日名のヘッダー行（soft塗り・下罫線つき）。時刻目盛りは1時間刻みで縦に並べ、各曜日列には角丸矩形の予定ブロックを配置する。ブロックは開始〜終了の時刻に応じて縦方向の行数をまたぐ（span）ことで高さが変わり、すべての時間帯を隙間なく埋める必要はない（空き時間があってよい）。

    structure:
      layout: grid (time-column + 5 day-columns)
      header-row:
        cells: 5 (月〜金, centered, soft fill)
        divider: bottom rule
      time-column:
        rows: hourly labels (e.g. 9:00〜17:00)
        align: left column, small text
      day-columns: 5 (月〜金)
      blocks:
        type: rounded rectangle with label
        span: variable (1〜複数時間分の行にまたがる)
        placement: 曜日ごとに複数個、隙間があってもよい
      gridlines:
        horizontal: thin dashed rule between hour rows
        vertical: thin rule at day-column boundaries

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 曜日ヘッダー | 最上段・5列（soft塗り） | 月〜金のラベルを示す |
| 時刻目盛り | 左端列・縦並び | 1時間刻みの時刻ラベル（9:00〜17:00など） |
| 予定ブロック | 各曜日列内・縦方向にspan | 時間帯ごとの予定・作業内容を角丸矩形＋ラベルで表示 |
| 空き時間 | ブロックのない行 | 予定が入っていない時間帯（余白として自然に表現） |
| グリッド罫線 | 行間・列間 | 時刻と曜日の境界を示す薄いガイド線 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-weekly-schedule-gridのレイアウトで、月〜金・[9:00〜17:00]の週間スケジュールを作成してください。各曜日に[2〜3個]の予定ブロックを、時間帯に応じた高さで配置してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 時刻目盛りは1時間刻みが基本。8〜9行（8〜9時間分）を超えると窮屈になるため、半日単位などに分割する。
- ブロックは曜日ごとに1〜3個程度が可読性の上限。詰め込みすぎない（すべての時間を埋める必要はない）。
- ブロックの高さは時間の長さに比例させる（30分と3時間の予定を同じ高さにしない）。
- ブロックのラベルは短い体言止め（例：「打合せ」「資料作成」）にし、長文を入れない。
