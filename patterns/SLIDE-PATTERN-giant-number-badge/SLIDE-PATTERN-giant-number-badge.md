# SLIDE-PATTERN-giant-number-badge

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** giant-number-badge
**概要：** 画面の1/3を占める特大数字1つ＋種別バッジ＋補足・出典。背景に超薄色の大円2つでリズムを付ける「1数字で語る」レイアウト。
**適したシーン：** 市場規模（TAM）・累計実績・達成率など、たった1つの数字を強烈に印象付けたい場面。ピッチの掴み。

## Structure（構造）
- 背景: 薄色の大円2つ（右上・左下。装飾はこれのみ）
- 左中央: 種別バッジ（塗り）→ 特大数字＋単位 → 補足1行 → 出典1行

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 種別バッジ | 数字の上 | 「市場規模」等のラベル（塗り＋白文字） |
| 特大数字 | 左中央 | 170px級。数字:単位:補足≒10:3:1のジャンプ率 |
| 補足・出典 | 数字の下 | 定義と根拠を小さく添える |
| 背景円 | 右上・左下 | 超薄色。テキストに被ってよい（opacity 0.55） |

## Usage Guide（AIへの使い方）
- 数字は1スライド1つだけ。複数あるなら three-kpi-big-number を使う
- 出典行は省略しない（数字の信頼性を担保する）
