# SLIDE-PATTERN-ascending-staircase-steps

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** ascending-staircase-steps
**概要：** 3〜4個の矩形カード（見出し＋短文）を右肩上がりの階段状に配置し、隣接するカードの間を「→」矢印でつなぐレイアウト。段階的な成長・レベルアップを表現する。
**適したシーン：** 成長フェーズ・成熟度の説明、キャリア・事業のステージアップ、レベル別の到達目標、右肩上がりのストーリーを視覚的に見せたい場面

## Structure（構造）

コンテンツエリア下部を基準に、3〜4個のカードを左から右へ順に高さ（下端からの距離）を段階的に上げて配置する。カード間には矢印を置き、視線が左下から右上へ流れるように誘導する。

    structure:
      layout: ascending-staircase
      items: 3-4
      item:
        type: card (heading + short-body)
        vertical-offset: increases stepwise per item (left low, right high)
      connector:
        type: arrow (→)
        position: between adjacent cards, offset roughly between the two step heights

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| ステップカード | 階段状に並ぶ矩形 | 各フェーズ・段階の名称と短い説明を表示 |
| 段差（縦位置） | カードごとに変化 | 右へ行くほど高くなることで成長・上昇のニュアンスを表現 |
| 矢印 | カード間 | 段階が右へ進行していく流れを示す |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-ascending-staircase-stepsのレイアウトで、[フェーズ1〜4]それぞれの見出しと短い説明を、右肩上がりの階段状に配置してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 項目数は3〜4個が上限。5個以上になると右端のカードが上部にはみ出しやすい。
- 見出しは短く（10字以内目安）、本文は1〜2行に収める。段差を作るぶん、カード内の情報量は少なめにする。
- 段差の大きさは均等にする（不揃いにすると階段に見えず崩れて見える）。
- 「右肩上がり＝ポジティブな成長」を表す構図なので、リスクや悪化など下降の内容には使わない（その場合は逆方向の階段か別パターンを検討する）。
