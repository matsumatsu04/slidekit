# SLIDE-PATTERN-summary-key-takeaways

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** summary-key-takeaways
**概要：** 特大の淡色番号＋アクセント短線＋要点＋補足を3カラム横並びにする「持ち帰り3点」まとめ。
**適したシーン：** セミナー・研修・提案の締め。今日の要点を3つに凝縮して持ち帰らせる場面。

## Structure（構造）
- 英字ラベル（KEY TAKEAWAYS）\n- 3カラム: 特大番号（淡色56px） → 短いアクセント線 → 要点（太字） → 補足2〜3行

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 特大番号 | 各列上 | 淡色で存在感だけ出す |
| 短線 | 番号下 | アクセント色の区切り |
| 要点 | 中央 | 太字1〜2行 |

## Usage Guide（AIへの使い方）
- 要点は必ず3つ（2つ・4つのときは summary-three-points を検討）
- チェックリスト調の内容なら summary-three-points、CTAで締めるなら next-step-cta-band へ
