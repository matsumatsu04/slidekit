# SLIDE-PATTERN-toc-two-column

このファイルはコンテンツエリアのレイアウト定義です。タイトル枠・ページ番号・装飾はデザインテーマ側の Frame で定義します。

## Overview
**パターン名：** toc-two-column
**概要：** 目次項目を左右2カラムに分けて並べる目次レイアウト。各行は大きな章番号＋セクション名＋ページ番号。
**適したシーン：** セクション数が5〜10個ある中長尺デッキの目次。1カラムでは縦に間延びする場合に。

## Structure（構造）
- header: 小ラベル（CONTENTS）＋目次タイトル
- grid: 2カラム × 3行（最大8項目程度）
  - item: 章番号（大・アクセント色）｜セクション名｜ページ番号（下線区切り）

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|---|---|---|
| 小ラベル | 左上 | CONTENTS等の英字ラベル |
| 目次タイトル | ラベル下 | 「目次」等 |
| 章番号 | 各行左 | 2桁ゼロ埋めの大きな番号（アクセント色） |
| セクション名 | 番号の右 | 太字 |
| ページ番号 | 行末 | p.n 形式 |

## Usage Guide（AIへの使い方）
- 項目は左列→右列の順（01-03が左、04-06が右）
- 項目数が4以下なら agenda-toc（1カラム）を推奨。5〜10項目でこのパターンを使う
